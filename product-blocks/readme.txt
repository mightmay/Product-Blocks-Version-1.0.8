Product blocks is a Gutenberg product blocks for WooCommerce, very useful for product listing, product grid and product slider.

== Description ==

Product Blocks is a Gutenberg product block plugin for WooCommerce, helping to create beautiful WooCommerce product grid, WooCommerce Product listing, WooCommerce product slider and WooCommerce Product carousel within a few seconds.

<strong>Live Editable Woo Product Blocks</strong>
Each and every setting is visually editable. You can change your design of the post with ultimate design settings. An enormous amount of google font has been added for your section design.

<strong>Product Slider</strong>
WooCommerce dynamic carousel using product is now easier to make using WooCommerce product blocks. WooCommerce product carousel will be run with your control, your desired design. Woocommerce Product Grid blocks come with ultimate control over the modern design.

<strong>Product Query Builder</strong>
Using WooCommerce Product Blocks query builder you can create a complex query for your product grid. Anyone can create complex queries for every block without writing a single line of code.

Yes, but it is a Gutenberg WooCommerce Block plugin. It displays the product grid or listing blocks on your page.
